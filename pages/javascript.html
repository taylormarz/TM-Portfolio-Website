portfolio website

I created a portfolio that would give me the opportunity to be experimental in both programming and design. I wanted to make something that was minimalist in nature, and allowed the projects on the website standout. I also wanted to challenge myself by using elements I hadn't tried yet in HTML and JavaScript. Specifically, I wanted to try and make an animation for drawings with the HTML Canvas element.
